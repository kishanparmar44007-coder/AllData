# 📊 Income Distribution and Household Demographics Analysis

## 📁 Dataset

This project uses a synthetic dataset of 150 households.

### Columns:

-   Household_ID
-   Age_of_Household_Head
-   Household_Income
-   Education_Level
-   Family_Size
-   Owns_House
-   Urban_Rural

## 📊 Analysis Performed

-   Central Tendency (Mean, Median, Mode)
-   Dispersion (Range, Variance, Standard Deviation, IQR)
-   Skewness and Kurtosis
-   Data Visualization (Histogram, KDE, Boxplot, Scatter Plot)

## 📈 Tools Used

-   Python
-   Pandas
-   NumPy
-   Matplotlib
-   Seaborn

## 🎯 Objective

To analyze income distribution and understand the relationship between
demographic factors and income.

## 📌 Conclusion

The project helps in understanding income variation, distribution shape,
and the effect of education and demographics on income.
