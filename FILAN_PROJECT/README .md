# 📊 Sales Analytics Dashboard (Power BI Project)

## 📌 Project Overview
This Power BI project analyzes **sales and return performance** using a well-structured **star schema data model**.  
It transforms raw transactional data into meaningful insights through interactive dashboards and visuals.

The dashboard helps users to:
- Track sales performance 📈
- Monitor returns 🔁
- Analyze customer behavior 👥
- Identify top products and regions 🌍

---

## 🛠️ Tools & Technologies Used
- Power BI
- Power Query (Data Transformation)
- DAX (Data Analysis Expressions)
- Excel / CSV Dataset

---

## 📂 Data Model
- Star Schema used:
  - Fact Table: Sales Data
  - Dimension Tables:
    - Customer
    - Product
    - Region
    - Date

---

## 🔄 Data Transformation (Power Query)
Following steps were applied:
- Removed null values
- Changed data types
- Created required columns
- Cleaned and structured dataset
- Followed proper ETL steps
