# 🎯 Expectation Decider Project

## 📌 Project Overview
This project uses **Probability & Statistics** concepts to predict whether a student will pass a mathematics exam.

The project includes:
- 📊 Probability Analysis
- 📈 Probability Distribution
- 🧠 Bayes Theorem
- 🔄 Conditional Probability
- 📋 Contingency Table
- 🎨 Venn Diagram
- 🐍 Python + Jupyter Notebook

---

# 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

---

# 📂 Dataset Features

| Feature | Description |
|---|---|
| study_hours | Weekly study hours |
| attendance | Attendance percentage |
| group_discussion | Participation in discussion |
| previous_test_score | Previous exam marks |
| final_exam_pass | Pass or Fail |

---

# 🚀 Project Workflow

## 1️⃣ Generate Dataset
Create student dataset using Python.

## 2️⃣ Probability Basics
Calculate:
- Empirical Probability
- Theoretical Probability

## 3️⃣ Distribution Analysis
Use Binomial Distribution to calculate passing probabilities.

## 4️⃣ Venn Diagram
Visualize overlap between:
- High Study Hours
- High Attendance

## 5️⃣ Conditional Probability
Find probability of passing based on group discussion.

## 6️⃣ Bayes Theorem
Predict passing chance with high attendance.
